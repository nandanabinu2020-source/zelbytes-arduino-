**Hardware Explanation**

Arduino Uno

The Arduino Uno serves as the central controller of the system. It generates PWM signals, reads emergency stop inputs, controls the relay module, and manages overall operation.

Motor Driver Module

The motor driver acts as an interface between the Arduino and the DC motor. Since the motor requires more current than the Arduino can provide, the driver amplifies the control signals and safely powers the motor.

DC Motor

The DC motor converts electrical energy into rotational motion. Its speed is controlled by varying the PWM duty cycle generated by the Arduino.

Relay Module

The relay module functions as an electrically operated switch. It allows the Arduino to safely control the 12V solenoid valve without directly handling the high-current load.

Solenoid Valve

The solenoid valve controls the flow of liquid. It opens when energized through the relay and closes when power is removed.

Emergency Stop Push Button

The push button provides an immediate shutdown mechanism. It is connected to Arduino interrupt pin D2, ensuring rapid response regardless of the current program execution.

LED Indicator

The LED provides visual feedback regarding system status:

ON → System active
OFF → Emergency stop activated
Resistors
220Ω Resistor: Limits current through the LED to prevent damage.
10kΩ Resistor: Acts as a pull-down resistor for the emergency stop button, ensuring stable input readings.
12V Power Supply

Provides power to the motor driver, DC motor, and solenoid valve while the Arduino handles control operations.
